########
Technote
########

Rubin Observatory's framework for Sphinx-based technote documents.

Install technote from PyPI:

.. code-block:: bash

   pip install technote

technote is developed on GitHub at https://github.com/lsst-sqre/technote.

.. button-link:: demo/index.html
   :color: primary

   View a demo technote page
